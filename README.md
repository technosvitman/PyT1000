# PyT1000
Python scriptable serial analyzer VT100 compatible

## Command line

Call *py pyT1000.py* followed by arguments

### List available ports

arg : _-list_
result : 
```
 ['COM3', 'COM4', 'COM12']
```

### Serial port parameters

#### Port

arg: _-p_
exemple : 
```
 -p COM12
```

#### Baudrate

arg: _-baud_
exemple : 
```
 -baud 115200
```
